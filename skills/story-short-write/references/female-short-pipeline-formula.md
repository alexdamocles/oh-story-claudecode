# 女頻短篇古言爽文 · 爆款流水線公式

> 來源：19 篇拆文分析大數據提煉（播放量 0–7637）
> 用途：配合 `/story-short-write` 使用，負責選題、粗綱、細綱階段

---

## 一、開篇鉤子（必選1種）

| 類型 | 經典場景 | 殺傷力 |
|------|---------|--------|
| **肉體羞辱型** | 交杯酒裡下絕子藥、安胎藥被換成墮胎藥、當街罰站兩個時辰 | ★★★★★ |
| **身份剝奪型** | 當眾奪誥命、廢后位、貶為側妃、搶嫁妝、毀名聲 | ★★★★☆ |
| **情感背叛型** | 新婚夜喊別人名字、為妾室罰正妻、放妻書遞來時她正繡戰袍 | ★★★★★ |
| **骨肉分離型** | 帶球跑被抓回、孩子被搶、為保胎被強灌藥 | ★★★★☆ |

> **法則**：鉤子必須涉及女性核心焦慮——**生育權、身體自主、尊嚴底線**。越具體、越視覺化，轉化率越高。

---

## 二、女主硬技能（必選1–2種）

| 技能 | 變現路徑 | 爽點場景 |
|------|---------|---------|
| **醫術** | 開醫館、入太醫院、專治婦科暗疾 | 前夫/新歡輾轉求到她門前排隊 |
| **刺繡** | 雙面繡、緙絲、入宮奉召、開繡莊 | 宮宴上她的作品碾壓全場 |
| **武力** | 女將軍、暗器、戰場、護衛鏢局 | 前夫被綁架時她去救 |
| **經商** | 茶樓、酒樓、繡坊、鹽運、錢莊 | 前夫來她的店消費卻不知東家是誰 |
| **廚藝** | 私房菜、藥膳、宴席設計 | 用菜名影射前夫罪行 |
| **琴棋書畫** | 入宮為師、名流雅集、開書院 | 太后/皇帝親自點名要她作品 |

> **法則**：「被拋棄」只賺同情，「被拋棄後我靠 X 手藝月入千兩」才賺爽感。

## 三、男主類型（必選1種）

| 類型 | 特徵 | 適用結局 |
|------|------|---------|
| **追妻火葬場型** | 前夫/前未婚夫，後悔莫及，付出真實代價 | BE 或開放式；若要 HE 須革職/天牢 |
| **新歡救贖型** | 從一開始就尊重她、欣賞她的才華 | HE |
| **側面守候型** | 默默幫助、後期才確認關係 | HE 或開放式 |
| **無 CP 型** | 女主獨美，不回頭，事業封頂 | BE / 開放式（近年高播放量趨勢） |

## 四、情緒節奏（黃金比例）

```
起 15-20% → 承 20-25% → 轉1 15-20% → 轉2 15-20% → 合 20-25%
```

| 段 | 情緒 | 必須完成 |
|---|---|---|
| **起** | 冰冷絕望 | 鉤子落地，女主反應立人設 |
| **承** | 沉靜蓄力 | 離開、技能變現、建立新秩序 |
| **轉1** | 溫暖希望 | 新男主登場 / 事業高峰，給口氣喘 |
| **轉2** | 緊張危急 | 前夫搗亂 / 陰謀揭露，推上高峰 |
| **合** | 釋然爽感 | 打臉報應 + 女主選擇 + 畫面定格 |

---

## 五、快速選題生成器

```
【鉤子】 + 【女主技能】 + 【變現路徑】 + 【結局類型】 = 選題
```

**範例組合**：
- `絕子藥` + `醫術/婦科` + `開醫館` + `不回頭` = 《麝香帳》
- `百子帳賞給新歡` + `緙絲/藏暗紋` + `入尚功局` + `終生未嫁` = 《裂帛書》
- `遞休書時繡戰袍` + `廚藝` + `開酒樓` + `HE嫁御史` = 《休夫宴》

## 六、爽點武器庫（至少用2個）

1. **前夫排隊求診**：她的店前夫不得不來，還得按規矩排隊遞名帖
2. **當眾打臉**：宮宴/雅集上，她的作品碾壓新歡/前夫現任
3. **藏證據於技藝**：繡品藏暗紋、菜品藏密語——技藝即武器
4. **以死相逼**：她用刀指著自己（而非前夫），逼他放人/認罪
5. **孩子認母**：帶球跑中，孩子當著前夫面只認她
6. **身份反轉**：棄婦實為郡主/公主/世家失散千金
7. **太后/皇帝撐腰**：高位者親自站台，前夫連跪的資格都沒有
