# 女频爽文虐渣：完整流水线模板

基于 19 篇高播放女频文（30,000-52,000字）逆向提取。适用于：伤害→隐忍→反杀→火葬场 四段式爽文。

---

## ⚠️ 受众校准（繁中/台湾女性听众）

### 核心原则

> **她承受的不是身体的暴力，是社会关系的暴力。她的反击不是"杀回去"，是"活给他看"。**

### 竞品数据说明的接受线

| 播放量区间 | 典型冲突 | 暴力烈度 | 代表篇 |
|:----------|:---------|:---------|:-------|
| 3.8万-11万 | 身份降级、冷暴力、被三 | 社会性羞辱 | 12年正妃降昭仪 |
| 1万-2万 | 公开羞辱、婚变+创业 | 情绪挤压 | 新婚夜去姨娘房 |
| 3千-1万 | 轻度涉及战争/大死伤 | **远景**战争 | 带兵赴边关 |

### 暴力呈现的镜头距离规则

| 暴力类型 | ✅ 可接受的呈现 | ❌ 不要的呈现 |
|:---------|:--------------|:-------------|
| **杀戮** | 远景（"三千将士覆没"）、结果（"他再也没有回来"） | 近景（"匕首划过喉咙""血溅了一脸"） |
| **刑场/处决** | 发生在视线边缘（"人群一阵骚动，有人捂住了孩子的眼睛"） | 发生在焦点（"头磕在木桩上，发出一声闷响"） |
| **生产/生育** | 孤独感和生存意志（"自己烧了水，靠着墙坐了一整夜"） | 生理细节（"剪脐带"的医疗器械描写） |
| **战争场面** | 谋略、决策、胜负结果 | 肉搏过程、伤口特写 |
| **死亡** | 消失、离去、不再出现 | 死状描写、血迹特写 |

### 代入成本公式

```
暴力画面距离 ÷ 读者共情难度 = 可接受的代入成本

19篇的公式：
  社会性羞辱（近） × 情感共鸣（高） = 高传播
  暴力杀戮（远） × 情感共鸣（高） = 高传播
  暴力杀戮（近） × 情感共鸣（高） = ❌ 消耗读者

不是"不能写"，是"不能写在视网膜上"。
让读者自己去脑补那层血——她们脑补出来的比写出来的更痛。
```

### 繁中女性听众的"爽感"构成

| 爽感来源 | 占比 | 说明 |
|:---------|:----:|:------|
| **她离开的姿态好看** | 40% | 不哭不闹不纠缠，沉默地消失最爽 |
| **他后悔了但晚了** | 30% | 火葬场不是物理死亡，是他"再也得不到" |
| **她活得比他好** | 20% | 事业/财富/地位上的碾压 |
| **他付出代价** | 10% | 代价不是死，是失去一切+永远活在悔恨里 |

**结论：这个赛道的听众要的不是"她杀了他"，是"她有多清醒地离开了他"。

### 反杀速度与爽感的关系

```
闪电反杀（半个时辰内）→ 当场爽 + 爆发力强（OlcLIrMs21I: 7,163）
   ↑ 适合：B/C级背叛
   
次日反杀（第2天）→ 让人意外 + 有布局感（DBzoS-TRclk: 3,023）
   ↑ 适合：A/B级背叛

布局反杀（数月→数年）→ 层次丰富 + 可扩展字数（22xwU3DIv80: 7,637）
   ↑ 适合：S/A级背叛（尤其是登基级反杀）
   ↑ 注意：这是唯一可以大幅扩写的反杀速度
```

### 反杀层级选用策略

**基本原则：背叛越狠，反杀要越高。** 匹配规则：

| 背叛等级 | 最低反杀等级 | 推荐反杀 | 理由 |
|:---------|:------------|:---------|:------|
| S级（杀子/灭门） | S级（登基/灭他满门） | SS1 / SS2 | 背叛太狠，D级反杀读者会骂女主窝囊 |
| A级（下毒/等死） | A级（假死/身份反转） | S1-S4 | 背叛已在死亡边缘，反杀需要一个"惊喜" |
| B级（休妻/羞辱） | B级（带走嫁妆/再嫁） | A1-A5 | 背叛是面子层面的，反杀到面子层面即可 |
| C级（利用/悔婚） | C级（归来打脸/揭穿） | B1-B5 | 背叛烈度中等，反杀突出一个"不是好惹的" |
| D级（冷落/忽视） | D级（离开/不回头） | C1-C3 | 背叛太弱，反杀太强反而违和 |

### 反杀动作的"黄金三要素"

每一个反杀场景必须同时满足：

```
① 有动作（不是"想"了，是"做了"）
   她不是"决定离开"，是"已经坐上了南下的船"
   她不是"想报复他"，是"让他的三千将士全军覆没"

② 有速度（越快越爽，但高级反杀允许慢）
   "半个时辰搬空家"比"三年后回来当女皇"更当场爽
   但"三年后当女皇"的后劲更大

③ 有结果（不可逆的）
   离开后不回来
   和离后不复婚
   登基后不退位
```

### 反杀场景的"必须镜头"

无论选哪种反杀类型，以下三个镜头必不可少：

| 镜头 | 功能 | 写法示例 |
|:-----|:-----|:---------|
| **他得知时的反应** | 逆转的第一次冲击 | 「管家跌跌撞撞跑进来：相爷——夫人她、她走了」 |
| **他追/找的过程** | 让读者享受"跟不上"的快感 | 「他翻遍了京城每一家客栈」 |
| **她离开的背影** | 画面定格，爽感沉淀 | 「船已行至江心，她没有回头」 |

### 选题策略

**新手首选（S/A级选一）：**
```
S1（杀子）+ A1（下毒）+ B1（休妻）→ 三重叠加，任何一篇都能到30K
这篇有19篇验证：22xwU3DIv80（S1 + 登王位 + 男主战死）播放7,637
```

**进阶（自创组合）：**
```
S2（灭满门）+ S4（杀子）+ 登基反杀 → 从未有过的组合，但逻辑必须通
```

**不推荐的组合：**
```
D级 × 3 = 无法撑起30K，读者看到标题就不会点
S级 + 低反杀 = 背叛和反杀不匹配，前面越重后面越要狠
```

| ⭐⭐⭐⭐ | 坦白两妾三子+挑衅 | 当场退婚+抬走嫁妆 | 傻眼→跪下 | vWs_RAWs0JY | 4,206 |
| ⭐⭐⭐⭐ | 新婚夜去姨娘房 | 当场递和离书 | 跪下求和 | 7YngQkyujd8 | 3,177 |

### 选题铁律

1. **不准有误会**：必须是实打实的伤害。不是"我以为他要害我"，是"他真的要害我/已经害了"
2. **不准原谅**：19篇中播放最高的几篇，没有一个和好的
3. **男主要付出代价**：越高播放，男主下场越惨

## 二、故事结构模板（5段式 + 字数分配）

### 总字数 30,000字 = 5段

| 阶段 | 功能 | 占比 | 字数 | 情节要求 |
|:-----|:-----|:----|:----|:---------|
| **一、伤害（开篇钩子）** | 男主做不可挽回的事 | 10% | 3,000 | 第一句话就进冲突 |
| **二、隐忍（铺垫蓄力）** | 女主表面接受+暗中布局 | 20% | 6,000 | 埋3-5条线索 |
| **三、反杀（实施行动）** | 女主离开/扳倒/反击 | 30% | 9,000 | 速度要快、层级要高 |
| **四、上升（扩写填充区）** | 反杀后果持续发酵 | 30% | 9,000 | **唯一可大幅扩写** |
| **五、火葬场（男主结局）** | 男主付出代价 | 10% | 3,000 | 下场越惨越爽 |

### 阶段一：伤害（3,000字）—— 开篇钩子

**第一句话必须是冲突**。对比：

| ❌ 错误写法 | ✅ 正确写法（19篇模式） |
|:-----------|:-----------------------|
| 「我嫁进赵府那天，天还没亮就被叫起来梳妆」| 「丈夫出征前，在我喝的安胎药里放了打胎药」|
| 「成婚五年，夫君把我送去庄子养病」| 「成婚第五年，夫君把我送去庄子养病，我摸着自己微微隆起的小腹，笑了笑」|

**开篇段必须包含的要素（前2,000字内完成）：**

1. 壹件已经发生或正在发生的伤害事件（下毒/休妻/送走/坦白背叛）
2. 女主的反应：**不哭不闹，接受**（反差越大越爽）
3. 至少一个贯穿道具首次登场
4. 男主的一个微表情破绽（"眼底一闪而过的如释重负"）
5. 旁人的反应（丫鬟急/下人偷笑/妾室得意）

### 阶段二：隐忍（6,000字）—— 埋线索

**表面写什么**：女主顺从/接受现实/看似认命。

**暗线埋什么**：

| 序号 | 线索类型 | 示例（来自19篇） |
|:-----|:---------|:----------------|
| ① | 物质转移 | 暗中转移嫁妆/银票/地契 |
| ② | 替代安排 | 换掉毒药/找好后路/安排替身 |
| ③ | 情报收集 | 找到把柄/账目/往来信件 |
| ④ | 身份准备 | 办新身份/改名/改户籍 |
| ⑤ | 道具准备 | 圣旨/兵符/婚书 |

**线索埋设规则**：
- 每一处线索都要**被读者看到但不被男主看到**
- 不能靠旁白交代，要靠女主的眼睛发现
- 线索密度：每1,000字至少1条

### 阶段三：反杀（9,000字）—— 速度决定爽感

**反杀速度与爽感成正比**：

| 反杀速度 | 示例 | 播放 |
|:---------|:-----|:----:|
| ⚡ **半个时辰** | 搬空家+带三万两出京城 | 7,163 |
| ⚡ **当天** | 当场递和离书+抬走嫁妆 | 4,206 |
| ⚡ **次日** | 圣旨和离+带兵赶赴边关 | 3,023 |
| 🐢 几年后 | 登王位/翻身回来（需要过程描写） | 7,637 |

**反杀行动链条（不可跳过任何一环）：**

1. **触发**：最后一根稻草（男主做了某件事让女主彻底死心）
2. **集结**：召集支持者（丫鬟/忠仆/娘家人/外部力量）
3. **执行**：动手（搬/带/签/走/烧/换——必须有**一个具体的大动作**）
4. **收尾**：离开现场（出门/上船/上马/上车——必须有**一个离开的画面**）

### 阶段四：上升（9,000字）—— ⭐唯一可大幅扩写的区域

**这是扩展字数的主要区域。** 19篇的中段全部在写同一类内容：**女主变强 + 男主衰落 + 外部世界展开。**

| 扩展方向 | 19篇示例 | 怎么扩展（不跑偏） |
|:---------|:---------|:-----------------|
| **事业崛起** | 登基后的朝政/收兵权/平叛乱 | 每一步都要**踩在他头上** |
| **男主惨状** | 下人汇报男主到处找她/发疯/跪求 | 用**第三视角写**，不要让她看见 |
| **新身份展开** | 成为女皇/将军/富商后的日常 | 写她掌控权力，不是写她怀念过去 |
| **复仇追加** | 清算当年帮凶/条人/旧敌 | 每一个被清算的人都是当年欺负她的 |
| **育儿线** | 抚养孩子/教孩子/孩子争气 | 如果是杀子梗，必须写**孩子活着且优秀** |

**扩展铁律**：
- ✅ 可以写：她执政/经商/掌兵/收买人心
- ✅ 可以写：他后悔/发疯/跪下/找人
- ❌ **不可以写**：她犹豫/心软/想他/动摇
- ❌ **不可以写**：他们见面产生暧昧
- ❌ **不可以写**：任何"其实他还爱她"的洗白

### 火葬场的关键画面（三幕式）

不管选哪种男主下场，都按三个步骤写：

| 幕 | 功能 | 怎么写 |
|:---|:-----|:-------|
| **第一幕：他知道了** | 男主意识到她走了/不爱他了 | 「管家跌跌撞撞冲进来：相爷——夫人她走了」「他推开房门，妆奁空了」|
| **第二幕：他追/找** | 让他付出"追不上"的努力 | 快马追到城门口/翻遍京城每一家客栈/在码头问有没有见过一个女子 |
| **第三幕：结果** | 彻底关上希望的门 | 船已至江心/她掀帘看了他一眼但没有停下/城门缓缓合上 |

**核心原则**：第一幕写他的震惊（给读者爽），第二幕写他的徒劳（给读者更爽），第三幕写不可逆的结果（给读者最爽）。每一幕都要有**一个具体的画面**——不要写"他很后悔"，要写"他跪在雨里，膝盖下的青石板湿了一片"。

### 火葬场检测清单

写完男主结局后逐条核对：

| 检查项 | 通过标准 | ❌ 常见问题 |
|:-------|:---------|:-----------|
| ☐ 他付出代价了吗？ | 有不可逆的损失（死/疯/跪/失去一切） | 他只是"后悔了"但没有失去任何东西 |
| ☐ 代价是她给的吗？ | 直接或间接因为她的反杀 | 他是"自己倒霉"不是"被她搞的" |
| ☐ 她亲手参与了吗？ | 最好由她亲手或她授意 | 全交给老天爷/命运/其他人 |
| ☐ 他认错了吗？ | 有他承认"是我错了"的时刻 | 他一直不觉得自己错了（读者会憋屈） |
| ☐ 但被原谅了吗？ | **没有被原谅** | 他说对不起她就说没关系 |
| ☐ 画面留住了吗？ | 有一个场景让读者记住 | 全通过对话交代，没有具象画面 |

## 八、镜头距离自查清单

繁中女性受众专用。每写完一个节点场景，对照检查：

### 暴力场景卡

| 检查项 | ✅ 通过标准 | ❌ 违规 | 修改方向 |
|:-------|:-----------|:--------|:---------|
| ☐ 杀戮发生在视线边缘 | 远景/他死了/他没回来 | 匕首割喉/血溅/断头声 | 那场仗打完，他没有回来 |
| ☐ 刑场/体罚不写身体 | 人群骚动/她低下了头/雪盖住了 | 断头落地/骨折/身体反应 | 人群一阵骚动，我没有看 |
| ☐ 战争不写伤痕 | 谋略/胜负/后果 | 伤口/残肢/肉搏过程 | 和她有关的战争描写应该在她的计策上了 |
| ☐ 伤害通过关系传递 | 他让别人递休书/他默许了 | 他亲手打了她 | 伤害的层级越间接，读者越容易代入 |
| ☐ 死亡是消失不是死状 | 他再也没有出现过/船走了 | 他死了/血流尽 | 「他消失在城墙上」比「他从城墙上跳下去」更有余韵 |

### 检查铁律

```
如果这一段让读者"看到了血"——删掉。
如果这一段让读者"感受到了她的痛"——保留。

读者不需要"看见"暴力，只需要"感受到"暴力的后果。
她们脑补出来的画面比任何描写都更精准。
```

## 九、贯穿道具与标题公式

### 贯穿道具规则

每一篇必须有 **1-2个贯穿全文的道具**：

| 道具类型 | 功能 | 示例 |
|:---------|:-----|:------|
| **定情物** | 前期温情→中期破坏→后期见证 | 簪子/玉佩/婚书/手镯 |
| **工具** | 中期反杀的实际工具 | 和离书/圣旨/兵符/账册 |
| **象征物** | 代表女主的转变 | 新身份/新衣/新簪子 |

**道具使用规则**：
1. 首次出现必须在**前段**（阶段一）
2. 功能转变必须**绕过男主**（他不该知道）
3. 最后一次出现必须在**结局**（阶段五）
4. 道具的状态变化 = 女主的状态变化

### 标题公式

```
[背叛事实] + [女主反应] + [反转结果] = 标题
```

| 公式分解 | 19篇示例 |
|:---------|:---------|
| 背叛事实 | 「丈夫在安胎药里放打胎药」「夫君以皇后之礼迎白月光」|
| 女主反应 | 「我不动声色换掉」「我当晚便乘马车离宫」|
| 反转结果 | 「四个月后我抱着儿子登上王位」「次日上門接人，岳父攔門」|

## 四、角色设定模板

### 女主设定

### 男主设定

| 维度 | 配置 | 铁律 |
|:-----|:-----|:-----|
| 性格 | 前期温柔/无情 → 后期疯魔/后悔 | **前期不能洗白** |
| 背叛行为 | 实打实的伤害 | 不准有苦衷（有苦衷就不是爽文了） |
| 专业词 | 心虚/不耐烦 → 后悔/疯魔/跪下 | 阶段一到阶段五的转变要有层次 |
| 下场 | 越惨越好 | 不准被原谅 |

### 配角设定

| 角色 | 功能 | 典型配置 |
|:-----|:-----|:---------|
| 丫鬟/忠仆 | 代表读者发声 | 会替女主着急、哭、骂男主 |
| 妾室/白月光 | 催化剂，推动背叛发生 | 得意→被连坐→下场也惨 |
| 娘家人（可选） | 反杀的支持力量 | 父亲/兄弟/忠心的管家 |
| 第三方势力 | 提供反杀工具 | 陛下/将军/商会/军队 |

## 五、扩写方法（不偏离主路线）

### 核心原则

> **不要在"感情线"上扩写，要在"权力线/事业线"上扩写。**

19篇的中段扩充全部是在女主变强这条线上做加法，不是在男主的感情线上做加法。

### 可扩写的内容清单（✅：可以用；❌：废了）

| 扩写方向 | 判定 | 为什么 |
|:---------|:----|:-------|
| 女主登基后的朝政细节 | ✅ | 展示她掌控权力，和男主无关 |
| 女主经商后的商业扩张 | ✅ | 展示她变强，和男主无关 |
| 男主的惨状（第三方视角） | ✅ | 写他后悔但**不让她知道** |
| 男主到处找她/跪求/发疯 | ✅ | 但是不能让她**心软** |
| 帮凶的清算 | ✅ | 当年欺负她的人被逐个干掉 |
| 孩子的成长/教育/反击 | ✅ | 让读者看到她的复仇获得了传承 |
| ❌ 她夜里睡不着想他 | ❌ | 纯废料，破坏人设 |
| ❌ 男主解释了当年另有苦衷 | ❌ | 第一个洗白就不能叫虐渣 |
| ❌ 她和他偶遇时心跳加速 | ❌ | 破坏反杀感 |
| ❌ 她犹豫要不要原谅他 | ❌ | 爽文女主的底线：不原谅 |
| ❌ 两人共同的朋友劝和 | ❌ | 废料剧情 |

### 扩写实操流程

```
写前问自己三个问题：
1. 我加的这段，是让女主更强了，还是让她更弱了？
2. 我加的这段，是让男主更惨了，还是让他更有希望了？
3. 这段去掉，主线会不会变？

如果第3题的答案是"不会变"——删掉，它是废料。
如果第1题的答案是"让女主更弱"或第2题的答案是"让男主更有希望"——删掉，它在破坏主线。
```

### 常见扩写陷阱

| 陷阱 | 识别方法 | 纠正 |
|:-----|:---------|:-----|
| 文青病 | 写长篇的内心独白/景物描写 | 每一句景物都要服务于情绪，否则不要 |
| 回忆杀 | 大量闪回从前美好的时光 | 回忆只能是"你是怎么对我的"，不能是"我们曾经多好" |
| 同情男主 | 写他失去女主后的痛苦太详细 | 略写。"他疯了""他跪了""他死了"——不要写他怎么哭 |
| 解释性对话 | 男主说"我当年其实是被逼的" | 直接删掉。读者不需要理由，只需要结果 |
| 和解式重逢 | 两人在某个场合见面，相视一笑 | 不要让他们见面。让他单方面跪/疯/死 |

## 六、完整写作流程

### 写作前（30分钟）

```
1. 选题矩阵选方向  →  2. 选反杀层级  →  3. 定男主下场
   (5分钟)              (5分钟)              (5分钟)
4. 定贯穿道具  →  5. 定标题公式
   (5分钟)          (10分钟)
```

### 写作中（按阶段）

```
阶段一 伤害：3,000字 → 阶段二 隐忍：6,000字 → 阶段三 反杀：9,000字
   (感官全面)        (暗线密度≥5条)         (速度优先)

阶段四 上升：9,000字 → 阶段五 火葬场：3,000字
   (权力线，非感情线)    (男主下场够惨)
```

每写完一个阶段，做一次感官密度自查 + 主线偏离自查。

### 写作后

```
感官密度检查 → 确认道具闭环 → 去AI味 → 完稿
```

## 七、与"治愈系言情"的对比

| 维度 | 爽文虐渣（本模板） | 治愈系言情（上次的测试篇） |
|:-----|:------------------|:-------------------------|
| 核心情绪 | **爽**——他活该 | 暖——终于和解了 |
| 背叛 | 真的伤害 | 误会/疑似 |
| 女主反应 | 提前布局→反杀 | 逃跑→自愈 |
| 反转方向 | 她比他更强 | 他没想害她 |
| 男主结局 | 付出代价（死/疯/跪） | 被原谅/和好 |
| 结尾 | 她独立/登基/消失 | 两人重新在一起 |
| 扩写方向 | 权力线/事业线 | 感情线/日常线 |

**两条赛道没有好坏之分，但选题时只能选一条，不能混用。**
